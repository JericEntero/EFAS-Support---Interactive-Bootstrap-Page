# EFAS Support Dashboard

Project Overview
This project is a responsive Dashboard, includes a Bootstrap Navigation bar, a feature card, a modal, and a validated form. It shows the basic UI design, responsiveness and form validation with Bootstrap 5.

---

Features

- Responsive Bootstrap Navbar (with hamburger menu)
- Dashboard Card section
- Bootstrap Modal integration
- Form inside Modal with validation
- Required fields validation (Name, Email, Message)
- Mobile-first responsive design

---

Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Bootstrap validation)

---

Responsiveness

The layout is responsive and tested with all of the following:
- Mobile view (375px)
- Desktop view (1200px+)

---

Form Validation Rules

A required field.A field that must be filled out.
- Email: (required) Must be in email format
A message that is required and must be of at least 20 characters.

---

Purpose

In this activity, you will learn how to create an interactive Bootstrap-based dashboard, including correct UI components and form validation.
